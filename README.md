# CIPH package
Just a simple package full of functions for encryption and decryption of multiple ciphers. Just for fun.

The package has the following ciphers:
- Numeric cipher
- Caesars cipher
- Polybius cipher
- Adfgvx cipher
- Bacon cipher
- Beaufort cipher

# Installation
Using devtools:
 
```
devtools::install_github("Red-Right-Hand1/CIPH_package")
```
